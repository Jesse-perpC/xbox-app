Background

It started with a simple file-picker for a mobile app builder solution in 2009. Over time we noticed that a solid file manager with various editors is a key component for a solid authoring software and so we coded it from scratch again. Over time we added support for external hosting like SFTP, FTP, Dropbox and others. It became our daily working tool and replaced our native FTP/SFTP clients.
Its now our mission to polish this application but also add rich file manager features such as a visual programming feature for automated tasks. Recent browser technology made it possible to bring native like user experience to the web. Its our goal to make much use of it as possible as our roots in developing rich authoring tools date back to 2000. In fact we re-invent and port what has been already there since decades, just without the limits of native development. Stay tuned, 'xfile' will land soon as stand-alone application for your operating system!

Roadmap (in that order)

  • Drop links into file panel for full download of the target page, or partial only. That feature will be fully customizable via the visual programming language called 'xblox'
  • Even better multi-media browsing experience
  • Sharing (Download manager)
  • P2P adoption ala Popcorn (use xfile for browsing & manage your p2p networks)
  • Visual interface for the internal & advanced permission system
  • Stand-alone executable for every operating system (currently at 70%)
  • Community driven extensions for the internal visual programming language
